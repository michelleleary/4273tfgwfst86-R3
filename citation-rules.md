# Правила цитирования ГОСТов

## Формат ответа-консультации

1. **Краткий вывод** (1–3 предложения)
2. **Нормативная ссылка**: `ГОСТ X.XXX-YYYY, п. X.X.X` (пункт — только если подтверждён источником)
3. **Редакция и статус**: действует / заменён на … / отменён (дата проверки)
4. **Источник**: URL официальной или надёжной базы

## При генерации ТЗ

- В теле ТЗ — только проверенные стандарты
- Служебный отчёт (при `service_report = true`) — полный перечень проверенных стандартов со статусами
- Без URL в теле ТЗ (только обозначение ГОСТ)

## Если данных нет

«Актуальность [ГОСТ] не подтверждена. Рекомендуется ручная верификация в [источник].»

## Групповые чаты

Отвечать только при @упоминании бота.
