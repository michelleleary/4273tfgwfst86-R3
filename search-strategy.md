# Стратегия поиска актуальных ГОСТов

## Порядок действий

1. Сформулируй обезличенный запрос (без названий заказчиков, шифров, коммерческой тайны)
2. Вызови search с приоритетом доменов:
   - `site:rst.gov.ru`
   - `site:docs.cntd.ru`
   - `техэксперт ГОСТ статус`
   - `кодекс ГОСТ действующая редакция`

## Примеры запросов

- `ГОСТ 15.016 статус действует 2026`
- `ГОСТ 2.103 действующая редакция`
- `замена ГОСТ 19.201-78`
- `ГОСТ 34.602-2020 отменён заменён`

## Проверка результата

- Сверь год редакции из двух источников при возможности
- Если стандарт заменён более 5 лет назад — используй только замену
- Зафиксируй дату проверки (текущая дата запроса)

## Если search недоступен

Не симулировать проверку. Сообщить пользователю и предложить ручную верификацию.

## Локальный корпус

Если файл есть в `/root/.hermes/knowledge/gost/` — приоритет локальной цитаты для точного текста пункта; актуальность статуса всё равно проверять через search.
